IPTV Playlist Checker

Описание:
Утилита для проверки IPTV-плейлистов (.m3u / .m3u8). Скрипт анализирует каждый канал, отправляет HEAD‑запрос и определяет, доступен ли поток. Поддерживается проверка локальных файлов и загрузка плейлиста по URL. Рабочие каналы сохраняются в отдельный файл.

Возможности:
- Проверка локальных плейлистов
- Загрузка плейлиста по прямой ссылке
- Многопоточность (по умолчанию 20 потоков)
- Цветной вывод статусов в консоль
- Автоматическое сохранение рабочего плейлиста
- Структурированная папка вывода

Установка:
Требуется Python 3.8+
Установите зависимости:
pip install requests

Запуск:
python playlist_checker.py

После запуска появится меню:
1. Выбрать локальный файл (.m3u/.m3u8)
2. Проверить по прямой ссылке (URL)

Структура вывода:
output/
 └── имя_плейлиста/
      └── checked_имя_плейлиста.m3u

Формат результата:
#EXTM3U
#EXTINF:-1,Channel Name
http://example.com/stream.m3u8

Как работает:
1. Скрипт читает плейлист построчно.
2. Находит пары: строка #EXTINF и следующая строка с http.
3. Для каждого URL отправляется HEAD‑запрос.
4. Если статус 200 — канал считается рабочим.
5. Все рабочие каналы записываются в новый файл.

Настройки:
В начале файла можно изменить:
TIMEOUT = 5      # Таймаут запроса
THREADS = 20     # Количество потоков

Пример вывода:
[OK] Discovery Channel
[DEAD] MTV (Status: 404)
[ERROR] Eurosport (Timeout/Error)
